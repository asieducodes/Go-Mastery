# Go-Mastery 🚀

> A structured, professional journey through Go backend development.
> Built alongside *Pro Go* by Adam Freeman.

---

## About This Repository

This repository is a systematic learning path for Go backend development.
Every chapter from *Pro Go* has a dedicated folder with:

- **examples/** — book examples retyped by hand
- **exercises/** — backend-focused exercises
- **challenges/** — stretch challenges for deeper understanding

---

## Learning Approach

```
1. Read the chapter in Pro Go
2. Retype the book example  →  examples/main.go
3. Solve the exercise       →  exercises/main.go
4. Attempt the challenge    →  challenges/main.go
5. Commit and push
6. Move to next chapter
```
